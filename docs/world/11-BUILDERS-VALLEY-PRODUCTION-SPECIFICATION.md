# Builder's Valley Production Specification

Defines the production-ready specification for the first playable environment.

## Scene layout
- Arrival Path
- River Corridor
- Bridge Crossing
- Workshop Terrace
- Waterfall Vista
- Village Edge

## Environment layers
Foreground: path, grass, fences.
Midground: bridge, workshop, river.
Background: waterfall, cliffs, mountains.

## Player flow
Arrival -> Explore -> First Build -> Cross Bridge -> Reach Workshop -> Return Loop.

## Gameplay readability
- Main path always readable.
- Landmarks visible from decision points.
- Lighting supports navigation.

## Runtime asset checklist
- Terrain
- River
- Bridge
- Trees
- Rocks
- Workshop
- Props
- Ambient audio

## Acceptance criteria
Composition matches approved Target Frame.
Traversal is intuitive.
Learning interactions are discoverable.
Performance target is maintained on supported devices.
