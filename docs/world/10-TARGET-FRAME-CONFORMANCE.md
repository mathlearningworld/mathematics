# 10 — Target Frame Conformance

## Purpose
Translate approved Hero Frames into measurable production targets.

## Builder's Valley checklist
- Camera composition and horizon locked.
- River corridor silhouette preserved.
- Bridge Crossing remains primary focal landmark.
- Waterfall readable from primary approach.
- Lighting validated for gameplay readability.
- Traversal path visible without UI.
- Vegetation supports composition instead of obscuring navigation.

## Runtime evidence
Each implementation must include:
1. Target Frame reference.
2. Runtime comparison screenshot.
3. Difference log.
4. Action items before acceptance.

## Acceptance
A production slice is accepted only when composition, gameplay readability, and learning intent all match the approved target frame.
