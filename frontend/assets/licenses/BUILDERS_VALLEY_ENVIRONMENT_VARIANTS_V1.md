## Ownership and origin

All assets in this pack are original project assets created specifically for Math Learning World. They were produced as assisted pixel-art vector assets and do not reuse marketplace previews, search-result images, third-party game assets, or identifiable artwork from another game.

## Intended use

This pack adds bridge structure, workshop storytelling, vegetation foreground, debris, and water-depth variants. Runtime integration is intentionally deferred to a separate composition slice so each asset can be reviewed before placement.

## Constraints

- Preserve nearest-neighbor rendering.
- Preserve upper-left light direction.
- Do not change gameplay, collision, placement, inventory, or Intent-Aware Hotbar behavior when integrating.
- Semantic asset IDs must remain independent from runtime implementation details.
